# 🍽️ Restaurant Website

A modern and responsive Restaurant Website built using HTML, CSS, and JavaScript. The website provides an engaging user experience with an attractive design, smooth navigation, and a fully responsive layout that works seamlessly across desktop, tablet, and mobile devices.

## 🚀 Features

* Responsive design for all screen sizes
* Modern and user-friendly interface
* Attractive hero section
* Navigation menu with smooth scrolling
* About Us section
* Food Menu section
* Contact section
* Interactive elements using JavaScript
* Clean and organized code structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

restaurant-website/

├── index.html

├── style.css

├── script.js

├── images/

└── README.md

## 💻 Installation

1. Clone the repository:

git clone https://github.com/eshasaleem09/restaurant-website.git

2. Navigate to the project directory:

cd restaurant-website

3. Open index.html in your browser.

## 🎯 Project Purpose

This project was developed to strengthen frontend development skills and demonstrate the ability to create visually appealing and responsive websites. It showcases modern web design principles and best practices in HTML, CSS, and JavaScript.
